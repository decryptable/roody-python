# roody-nodejs
A conversion of the "Roody" bot from Go to Node JS

## Installation

Windows:

```
# Installing modules
pip install discord
pip install asyncio
pip install bs4

# Running
python roody.py
```

Linux:

```
# Installing modules
pip install discord
pip install asyncio
pip install bs4

# Running
/usr/bin/python3.5 roody.py
```
