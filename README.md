# Shakespeare -- Python Port
A clone of, a now defunct insult bot, "Roody" which insults you.

## Installation

Windows:

```
# Installing modules
pip install discord.py
pip install asyncio
pip install bs4

# Running
python roody.py
```

Linux:

```
# Installing modules
pip install discord.py
pip install asyncio
pip install bs4

# Running
/usr/bin/python3.5 roody.py
```
